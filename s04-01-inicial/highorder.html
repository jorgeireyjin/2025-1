High Order Functions

Concepto

Las funciones de orden superior (high Order) son funciones que utilizan funciones como argumento o valor de retorno. Son una característica importante en la programación funcional, ya que permiten anidar funciones para lograr una tarea específica.

Algunas funciones ejemplo en JS:

  • reduce : El método itera a través de un arreglo y devuelve un valor único.
  • forEach : Recorre cada uno de los elementos de una matriz en orden.
  • map : Crea un arreglo con los resultados de invocar a una función "callback" proporcionada como argumento.
  • filter : ejecuta una función callback en cada elemento de un arreglo. La función callback devuelve V o F por cada elemento. Se devuelve un arreglo solo con los elementos que retornaron V
