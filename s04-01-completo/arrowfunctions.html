Arrow Functions

Una "arrow function" (o función flecha) en JavaScript es una forma concisa de escribir funciones. Introducidas en ECMAScript 6 (ES6), las funciones flecha tienen una sintaxis más corta, lo que las hace útiles en ciertos casos, como en callbacks y programación funcional.

Sintaxis básica


            const nombreFuncion = (param1, param2) => {
                 // cuerpo de la función
            };

Si la funcion tiene un solo parametro:


            const nombreFuncion = param => {
                // cuerpo de la función
            };

SI la funcion tiene una sola expresión se puede omitir las llaves y el return


            const suma = (a, b) => a + b;
